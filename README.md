Online memory card game themed with the team of [Indigo Tree Digital](https://indigotree.co.uk/meet-the-team/) "inspired" (ie mostly copied) by [Tania Rascia's version](https://github.com/taniarascia/memory). Thanks Tania üëç

## Instructions

- Duplicate the cards to have 2 sets of 14.
- Randomize the display of cards.
- Show back of card initially and flip on select.
- Add selected style for selected cards.
- Only allow two cards to be selected at once.
- Determine if two selected cards are a match and hide them.
- Reset guess count after 2.
- Add delay to selections.
- Finished game!
